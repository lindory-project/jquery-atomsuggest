AtomSuggest

Aide à la saisie à partir d'un flux ATOM/OpenSearch

Installation

Ce plugin requis la librairie Javascript JQuery. Vous pouvez la télécharger sur le site officiel de JQuery.

Première étape

La première étape consiste à inclure le fichier jquery.js dans l'élément "head" de votre page Web, suivit par le fichier jquery.atomsuggest.js.

REMARQUE: Le fichier jquery.js doit se placer en premier dans l'ordre des références aux ressources externes. 

<script type="text/javascript" src="jquery.js"></script>
<script type="text/javascript" src="jquery.atomsuggest.js"></script>

Deuxième étape

La deuxième étape consiste à inclure les fichiers CSS dans votre page Web. Il y en a deux :

  • atomsuggest.grey.css qui gère les couleurs du plugin.
  • atomsuggest.css qui gère toute la structure du plugin (écriture, taille, dimmensions, marges, etc.).

Il vous suffit d'inclure ces fichiers dans le "head" de votre page Web, comme dans l'exemple suivant : 

<link rel="stylesheet" type="text/css" href="style/atomsuggest.css" />
<link rel="stylesheet" type="text/css" href="style/atomsuggest.grey.css" />

REMARQUE: Vous pouvez modifier les règles css selon votre convenance (couleur, police, etc.).

Troisième étape

La dernière étape d'installation consiste à implanter dans votre page un script JavaScript permettant de définir des options personnalisant le plugin, et d'indiquer le champ à auto-compléter. Ce script doit être placé dans la partie "head". 

$(document).ready(function() {
    var options = {
        "url": "http://localhost/testSuggestTermSciences/tsproxy.php",
        "widthFactor": 2.5,
        "showInfos" :  { "identifier" : true, "updated" : true, "content": true", "result" : true },
        "paramValue": "query", 
        "extraParams":  { "ws": "search", "alt": "json", "lang": function() { return $("#lng").val() } },
        onSelect: function(entry) { alert(entry.id.value); $('#searchform').submit();}
    }
    $("#searchform input[name='qry']").atomsuggest(options) ;
}

Ce plugin gère uniquement les sources de données au format JSON et XML Atom.

REMARQUE : Les options précédées par un point noir sont le minimum permettant le fonctionnement du plugin.

Signification des options :

Nom de l'option Signification
"url" Chemin de votre script proxy.
"widthFactor" Multiplicateur de la largeur de l'élément regroupant les suggestions. Par défaut la largeur correspond à la largeur de l'input concerné, le multiplicateur vaut donc 1.
"showInfos" Permet de configurer des éléments supplémentaires à afficher ( "identifier": identifiant de chaque terme; "updated": date de dernière modification; "content": description du terme; "result": nombre de résultat répondant à la saisie). Par défaut ces informations ne sont pas affichées.
"paramValue" Nom du paramètre qui contiendra la valeur saisie (dans l'exemple, "query" correspond au nom du paramètre du web service utilisé permettant de récupérer les données répondant à la saisie).
"extraParams" Ses valeurs correspondent aux paramètres concernant les web services alimentants le plugin. A gauche, le nom du paramètre et à droite, sa valeur.
"onSelect" CallBack sur l'élément sélectionné. Elle permet notamment de poursuivre des traitements sur l'élément sélectionné.
"onRequest" CallBack sur les caractères saisies. Elle permet de modifier la requete juste avant d'envoyé la requete au serveur. 
$("#qry").atomsuggest(options)

Cette ligne de code permet d'appeler tout le traitement permettant l'auto-complétion. Il vous faut indiquer en sélecteur l'identifiant du champ "input" à auto-compléter. Dans l'exemple l'identifiant est "qry"

Vous pouvez également indiquer en paramètre le nom de votre "input". Dans l'exemple le nom du champ est "qry" 

$("#searchform input[name='qry']").atomsuggest(options)

REMARQUE : Il vous faut donc impérativement un élément "input" dans votre page Web ainsi qu'un identifiant caractérisant votre "input".

Exemples

L'utilisation est très simple, une fois votre plugin branché sur votre champ texte, il suffit de saisir des informations et vous verrez la liste des suggestions apparaitre automatiquement en fonction de votre saisie.
